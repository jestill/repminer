NAME

cnv_tab2fasta.pl - Convert tab delim text file to fasta format

VERSION

This documentation refers to program version $Rev: 100 $

SYNOPSIS

Usage

    cnv_tab2fasta.pl -i infile.txt -o outfile.fasta

Required Arguments

    -i        # Path to the input text file
    -o        # Path to the output fasta file

DESCRIPTION

Converts a tab delimitted text file into the fasta format. The last column in the tab delimited text file is assumed to be the sequence data. Preceeding columns are added to the fasta header file separated by the pipe delimiter. It is possible to have variable number of columns for each sequence record, as long as the last column is always the sequence string. The typical use for this program will be to convert a text dump for a database or Microsoft Excel to a fasta format text file.

REQUIRED ARGUMENTS

-i,--infile

Path of the tab delimited text input file. If a file is not specified by the --infile option, the program will attempt to read input from STDIN.

-o,--outfile

Path of the fasta format output file. If a file is not specified by the --outfile option, the program will write output to STDOUT.

OPTIONS

--usage

Short program description of usage.

--help

Show program usage with summary of options.

--version

Show program version.

--man

Show the full program manual. This uses the perldoc command to print the POD documentation for the program.

-q,--quiet

Run the program with minimal output.

EXAMPLES

Typical Use

In the typical use case, you will want to convert a tab delimitted text file produced from a database dump to the fasta file format

 cnv_tab2fasta.pl -i infile.txt -o outfile.fasta

This will convert an intput text file in the following format

 seq_1   maize   chrom_1 ATGCAA...
 Seq_2   maize   chrom_1 GCCATC...
 seq_3   sorghum chrom_1 TCAGCC...

To the following format

 >seq_1|maize|chrom_1
 ATGCAA...
 >seq_2|maize|chrom_1
 GCCATC...
 >seq_3|sorghum|chrom_1
 TCAGCC...
